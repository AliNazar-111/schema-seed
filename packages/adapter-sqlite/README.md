# schema-seed-adapter-sqlite

Sqlite adapter for [schema-seed](https://www.npmjs.com/package/schema-seed).

## Installation

```bash
npm install schema-seed-adapter-sqlite
```

## Usage

This adapter is automatically loaded by the `schema-seed` CLI when a sqlite connection string is used.

For more information, visit the [main schema-seed documentation](https://github.com/AliNazar-111/schema-seed).
